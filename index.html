Google 登入 — OAuth Client ID

前往 Google Cloud Console → API 和服務 → 憑證 → 建立 OAuth 2.0 用戶端 ID（類型選「網頁應用程式」），將您的網站網址加入「已授權的 JavaScript 來源」後複製 Client ID 貼入。

Facebook 登入 — App ID

前往 Facebook 開發者平台 → 建立應用程式 → 新增「Facebook 登入」產品，在「網頁」設定中填入您的網站網址，再複製 App ID 貼入。

Gemini API 金鑰
