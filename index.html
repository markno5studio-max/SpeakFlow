字幕內容

macOS 安全政策封鎖直接開啟字幕檔。請使用以下任一方式取得檔案：

① 複製全部 → 開啟「文字編輯」→ 貼上 → 儲存 → 副檔名改 .srt
② 或下載 .txt → 開啟 → 另存新檔 → 將副檔名改為 .srt
③ 在「終端機」貼上以下指令移除隔離屬性，即可正常開啟已下載的 .srt：
匯出字幕

請選擇字幕檔案格式：

純文字 SRT（標準）
業界標準格式，相容 Final Cut Pro、Premiere Pro、DaVinci Resolve 等所有專業剪輯軟體。字幕外觀（含背景色）由軟體設定控制。
ASS 無黑底字幕
進階字幕格式，白色文字加黑邊輪廓，無黑底方塊。適用於 DaVinci Resolve、VLC、mpv 等支援 ASS 的播放器與軟體。

關於 FCP 黑底：Final Cut Pro 字幕軌預設顯示黑底，這是 FCP 的字幕渲染設定，非由 SRT 檔案控制。可在 FCP「字幕角色」中修改樣式。

Google 登入 — OAuth Client ID

前往 Google Cloud Console → API 和服務 → 憑證 → 建立 OAuth 2.0 用戶端 ID（類型選「網頁應用程式」），將您的網站網址加入「已授權的 JavaScript 來源」後複製 Client ID 貼入。

Facebook 登入 — App ID

前往 Facebook 開發者平台 → 建立應用程式 → 新增「Facebook 登入」產品，在「網頁」設定中填入您的網站網址，再複製 App ID 貼入。

Gemini API 金鑰（Google AI Studio）

前往 Google AI Studio (aistudio.google.com) 取得 API 金鑰，啟用 Gemini 2.5 Pro / Flash 雲端轉錄。

OpenAI API 金鑰（Whisper / GPT）

前往 platform.openai.com 取得 API 金鑰，填入後即可使用 OpenAI Whisper-1 雲端轉錄（逐字稿精準度高，支援多語言）。金鑰僅儲存於本機，絕不上傳。

本地 Whisper 模型大小

更換模型後需重新下載。模型支援繁體中文、英文、日文等多語言。
